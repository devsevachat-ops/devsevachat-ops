# 👋 Hi, I'm the builder behind Sevachat

## About Sevachat

Sevachat is an AI receptionist platform for salons that handles:

- WhatsApp conversations
- Appointment booking
- Customer support
- Lead qualification
- Multilingual communication
- 24/7 availability

## Tech Stack

- Next.js
- TypeScript
- Node.js
- PostgreSQL
- WhatsApp Business API
- OpenAI
- Docker
- GitHub Actions
